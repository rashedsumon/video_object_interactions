# Video Object Interaction Analysis

## Description
This project detects multiple objects in a live video stream and analyzes the interactions between four specific objects. It generates visual representations and rule-based insights, enabling monitoring, decision-making, or further AI applications.

## Features
- Real-time object detection and tracking.
- Visualization of interactions between four target objects.
- Rule-based interaction analysis.
- Statistical summaries and charts.
- Logs of interactions.
